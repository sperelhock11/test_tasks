В проекте очишены кеши логи асеты и удалена папка vendor.
-------------------------------------------------------------------------
Для запуска проекта нужно 
запустить докер 
docker compose up
потом зайти в контейнер
docker exec -it test_task-php-1 bash
и в нем запустить 
composer update
после этого папка vendor появится и проект должен заработать
